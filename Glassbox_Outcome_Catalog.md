## Overview

Glassbox offers 10 productized engineering outcomes across four categories: **Build**, **Automate**, **Optimize**, and **Migrate**. Each outcome is a fixed-scope, fixed-price engagement delivered by AI-native engineers at 2-3x the velocity of traditional teams. Every engagement includes a dedicated Project Delivery Lead, real-time dashboard access, milestone-based delivery, and a 30-day post-delivery warranty.

## Engagement Model

All outcomes follow the same delivery framework:

1. **Intake** — Client fills out a structured questionnaire specific to the outcome type
2. **Scoping** — FullStack PM reviews, clarifies, and confirms scope within 24 hours
3. **Kickoff** — Engineering team assembled, architecture finalized, sprint planned
4. **Build** — AI-accelerated development with daily progress on the Glassbox dashboard
5. **Review** — QA pass, client review, revisions
6. **Delivery** — Production deployment, documentation, handoff
7. **Warranty** — 30-day post-delivery support period

Every engagement includes:
- Dedicated AI-Native Project Delivery Lead
- Real-time Delivery Dashboard (milestones, code activity, health score)
- Daily EOD reports from the PM
- Read-only access to the live codebase
- Full project documentation repository
- Communication hub for async discussion
- Fixed price — no scope creep surprises
